Overview

The Web DICOM Editor lets you review and edit DICOM images entirely in your browser. Load a folder of scans, explore each series, adjust windowing, and edit pixels using painting and selection tools. All edits stay in memory until you save them to an output folder of your choice.

Loading DICOM Data

  • Click Load DICOM Folder to pick the input folder. The application scans all nested folders and loads any file that contains the DICOM signature.
  • Choose Output Folder to tell the editor where to store saved files.
  • The status message in the header confirms how many series were discovered or reports any issues.

Series Navigation

  • The left panel lists every series with a thumbnail, number, and description.
  • Click a series card to load its images into the viewer.
  • Use the mouse wheel, trackpad scroll, or the ↑ / ↓ keys to move through slices ordered by InstanceNumber.
  • The information overlay in the viewer shows slice details, windowing, and zoom.

Viewer Tools

Select a tool from the right-hand toolbar to control how pointer drags affect the view.

Window (W)
Drag horizontally to adjust window width and vertically to adjust window level.
Pan (P)
Drag to reposition the image when zoomed in.
Zoom (Z)
Drag up to zoom in and down to zoom out. Scroll wheel zooming is also supported.
Brush (B)
Paint directly onto the active slice using the brush size and intensity controls.
Select (S)
Drag to create a selection volume. Scroll while dragging to extend the selection above or below the starting slice.
Save
Click to write all modified images to the chosen output folder.

Brush Controls

When the brush tool is active, a floating panel appears in the top-right corner of the viewer.

  • Brush Size sets the diameter in pixels of the paint stroke.
  • Intensity controls the grayscale value written to the DICOM data.
  • The numeric readouts update live as you adjust the sliders.

Selection & Copy / Paste

  • Use the Select tool to drag out a rectangular selection on the current slice.
  • Scroll during the drag to extend the selection through adjacent slices. The highlighted volume updates as you establish the upper and lower bounds.
  • Press Ctrl + C to copy the selected volume to the clipboard.
  • Navigate to another slice or series. The paste preview appears centred on the current slice and can be dragged before committing.
  • Click to position the preview, then click outside the overlay to merge the volume into the underlying images. The slice you paste on becomes the midpoint of the applied volume along the Z axis.
  • If you change series, both selection and paste previews reset to avoid conflicts.

Saving Changes

  • Click Save or press Ctrl + S to export edits.
  • All modified instances are written to the output folder using their original file names and directory structure.
  • The header status message lists how many files were saved, and the unsaved changes indicator clears once the operation completes.

Troubleshooting

  • If nothing appears after loading, confirm the selected folder contains valid DICOM files and that the browser has permission to read them.
  • Use the browser console to review diagnostic logs for loading, tool changes, and selection updates.
  • Re-select the output folder if saving fails—browsers can revoke directory access after page reloads.
